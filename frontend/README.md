# frontend

## Project setup
- Setup Nodejs version 16 if any
- Open terminal on your code editor
- Navigate to this file's directory and type some commands below

### Type this when your code does not have 'node_modules' folder
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Recommend not running this if there are some issues in front-end side
```
npm audit fix --force
```
